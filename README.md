# Aplikacja rezerwacji apartamentu

To jest startowy szkielet aplikacji do obslugi rezerwacji apartamentu
krotkoterminowego.

## Co juz zawiera projekt

- Next.js z TypeScript
- podstawowy widok publiczny
- placeholder panelu administratora
- placeholder strony platnosci `/pay/[token]`
- testowy endpoint `GET /api/health`
- katalogi pod baze danych, uslugi, integracje, e-maile i testy

## Najblizszy krok

Dodanie modeli danych w Prisma oraz logiki sprawdzania dostepnosci terminow.
